Checklist for Each Page

  • The link supplied goes straight to the page being reviewed.
  • No filenames contain spaces or uppercase letters.
  • Design
    • Text contrast and font sizing are easy to read.
    • Colors and fonts come from the shared default.css.
    • CRAP principles are visible: contrast is clear, styles repeat consistently, elements align neatly, and related items are grouped closely.
  • Page contains a <header>, <main>, and <footer>; main navigation sits between header and main.
  • The header holds the site / brand name inside an <h1> element.
  • The top of main begins with the page title in an <h2> and does not duplicate the brand name.
  • The brand tagline appears in a consistent location on every page.
  • The footer lists the user’s pages and includes working HTML and CSS validation links that pass.
  • All specific Client-Project Pt. 2 items—client goals, timeline, personas, and wireframes—are in place.
